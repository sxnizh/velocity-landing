# Velocity 
Небольшой учебный лендинг с адаптивной версткой и табами на JavaScript. Проект сделан на чистом HTML, CSS и JavaScript без сборки. 

## Демо 
https://sxnizh.github.io/velocity-landing/ 

## Технологии 
- HTML
- CSS
- JavaScript 

## Что есть в проекте
- фиксированная шапка
- плавный скролл по якорям
- адаптивная верстка
- секция с карточками
- секция services
- табы на JavaScript
- футер с ссылками и соцсетями

## Как запустить 
1. Клонировать репозиторий
```bash
git clone [https://github.com/sxnizh/velocity-landing.git](https://github.com/sxnizh/velocity-landing.git)
```
3. Открыть папку проекта

```bash
cd velocity-landing
```
5. Открыть index.html в браузере
